Constructors


Used to reset an object member variables when the object is first created
If a constructor has parameters, they must be passed
Classes can have many different constructors, only requirement is that each one has different parameters or types
If no constructor is defined, then C++ will generate an implicit default constructor (does not initialize primitive member variables and they will have uncertain values)
Constructors are called each time a variable for a class is created
Constructor called N times when an array of size N is created
