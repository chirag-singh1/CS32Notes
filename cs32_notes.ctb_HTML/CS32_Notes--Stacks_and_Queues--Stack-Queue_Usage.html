stack<int> s; //Creates empty stack

}

//Goal: evaluate expression 
//Use mathematical prefix notation: f(x,y,z) + - 8 / 6 2 1
//Infix notation: 8-6/2+1 (more ambiguous)
//Postfixnotation: 8 6 2 / - 1 +  (less ambiguous)
//Goal of algorithm: Convert to postfix notation, then process 
//Algorithm to convert to postfix:
/**
If current item operand, append to result sequence
If current item is (, push to stack
If current item is ), pop operators off stack and append until (
If current item is operator:
If stack empty, push to stack
If top of stack is (, push to stack
If current operator greater precedence than top of stack, push to stack
Else
Pop the top operator from the stack and append to result and check again
At end of sequence, pop each operator off stack and append to result seuqnece

**/
//Algorithm to evaluate postfix: Push each operand to operator stack, when operator found apply operation to top 2 in the stack, then push result to stack
